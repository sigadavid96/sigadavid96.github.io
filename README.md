## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sigadavid96/sigadavid96.github.io.git
cd sigadavid96.github.io
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:4321/`

## ğŸš€ Deployment

This portfolio is deployed using GitHub Pages. To deploy your own version:

1. Update the site configuration in `src/config.ts`
2. Build the project: `npm run build`
3. The `dist/` folder contains the static files ready for deployment
4. Push to the `main` branch of your GitHub repository named `[username].github.io`
5. Enable GitHub Pages in repository settings (if not already enabled)

The site will automatically deploy to `https://[username].github.io/`
